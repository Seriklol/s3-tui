# s3-tui

A small terminal user interface for managing buckets and objects in S3 compatible storage.

## Installation

### Arch Linux

```sh
git clone https://github.com/Seriklol/s3-tui.git
cd s3-tui && makepkg -si
```

## Features
- Navigation and management of available buckets
- Download/upload/deletion of files
- Ability to resume partial downloads
- Ability to resume or cancel partial uploads
